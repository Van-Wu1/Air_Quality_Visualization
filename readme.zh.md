## 🌟 主要功能

✅ **实时 AQI 数据** - 监测全球 81 个城市的空气质量  
✅ **7 天 AQI 预测** - 提供未来一周的空气质量趋势  
✅ **数据可视化** - 交互式地图和图表展示空气污染情况  
✅ **开放讨论** - 欢迎就可视化方式提出你的想法！
