About This Project

Global Air Quality Visualization is an interactive web platform that provides real-time air quality data and 7-day forecasts for global 81 cities. This project aims to help users understand air pollution trends through interactive maps and charts.

⭐Author's brain dump: This project originated from an assignment in the CASA 0003 course. In fact, it was somewhat overworked, but the author's beautiful mental state（留子美丽的精神状态🥴） kept her tirelessly working on the frontend without rest. And she found front-end design as addictive as a drug. She even set up a studio for herself.

Key Features

  • 📌 Real-Time AQI Data - Get live air quality data for 81 cities worldwide.
  • 📊 7-Day PM2.5 Forecast - Predict air pollution trends to plan your activities.
  • 🗺️ Interactive Map - Visualize global AQI with an intuitive map even roaming the continent.
  • 🔍 Report export - More detailed air quality data can be downloaded.
